# Salesforce DX Project

## Overview

This repository contains a Salesforce DX project demonstrating environment setup and Git workflow.

## Tools Used

- VS Code
- Salesforce CLI
- Salesforce Extensions for VS Code
- Git
- GitHub

## Setup Steps

1. Installed Salesforce CLI.
2. Installed VS Code and Salesforce Extensions.
3. Authenticated to Salesforce Org.
4. Retrieved Apex Classes and Flows.
5. Initialized Git repository.
6. Created feature branch.
7. Committed changes.
8. Merged feature branch into main.
9. Pushed repository to GitHub.
